Inverite AgeVerify plugin for Wordpress and WooCommerce to verify a customer's Age and Identity

== Description ==
If you are selling an age-restricted product or simply need to confirm your customer's identity for KYC/compliance purposes, this plugin allows painless integration with Inverite's AgeVerify service.  Customer's will be presented with a modal dialog to upload identity documents that will be stored in their Customer for future purchases.

Several different methods are supposed including thankyou, checkout, and profile page bindings.

The plugin can also be configured with different ID requirements all the way from facial recognition to simply parsing the back of the ID.

All Canadian Driver's Licences, Provincial Identification Cards and Provincial Services and most US Driver's Licenses and ID cards are supported.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Sign up for an Inverite account though https://www.inverite.com/
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
